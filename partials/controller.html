Routing with a controller

When you route a request to a template you can also specify a controller.
Doing so you can add extra individual control of content on the template based pages.
This path have a controler, which fills out the values here:

  • params:{{params}}
  • loc: {{loc}}

function ControllerCntl($scope, $routeParams) {
  $scope.name = "ControllerCntl";
  $scope.params = $routeParams;
  $scope.loc = document.location.href;
}
